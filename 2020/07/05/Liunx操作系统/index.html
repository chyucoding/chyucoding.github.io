1. 为什么选择Liunx 作为操作系统？

开源免费

相比于windows更加安全 更加稳定 据说十年可以不关机依旧流畅运行；

2. Liunx常用命令：
  • cd 进入某一个目录
    • cd ~进入用户目录
    • cd / 进入磁盘根目录
    • cd .. 返回上一级
  • pwd 显示当前所在路径的根路径
  • clear 清屏
  • ls 以平铺的方式显示当前目录所包含的内容
  • ll 以列表的形式显示当前目录所包含的内容
  • ls -la 以列表的形式显示当前目录所包含的内容以及隐藏的内容
  • mkdir 新建一个空文件夹 语法：mkdir -p 路径：如果路径中包含了不存在的路径，则自动创建；
